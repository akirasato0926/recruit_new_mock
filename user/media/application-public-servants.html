公務員試験の申込み方法 ── 受付期間と手続きの流れ

PUBLIC CONNECT 編集部2026.06.08

受験の第一歩は申込みです。受付期間が短い試験も多いため、早めの準備が肝心です。

目次

  1. 申込みは電子申請が主流
  2. 受付期間に注意
  3. 受験票の取得

申込みは電子申請が主流

近年はインターネットからの電子申請が一般的です。顔写真データや必要事項を準備し、受付期間内に登録します。

パブリックコネクト等の求人サイトから、募集情報をまとめて確認することもできます。

受付期間に注意

受付は数週間と短いことが多く、締切間際はアクセスが集中します。余裕を持って手続きしましょう。

受験票の取得

申込み後、受験票がマイページ等で発行されます。試験当日まで大切に保管してください。
